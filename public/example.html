This is a demo page showing how the Apptega Navigator Widget integrates with your application. The widget is displayed in the sidebar on the right.

Method 1: Direct Iframe Integration

The simplest way to integrate the navigator widget is using a direct iframe embed:

<iframe src="http://localhost:3000/" width="400" height="600" frameborder="0" id="apptega-navigator"> </iframe>

This method is used in the sidebar on the right side of this page.

Method 2: Script-based Integration

For more advanced integration with communication between the widget and your app:

<div id="apptega-navigator-container"></div> <script src="http://localhost:3000/embed.js"></script> <script> AppregaNavigator.init({ container: '#apptega-navigator-container', userRole: 'organization', organizationId: 'org_123', userId: 'user_456', onNavigate: function(path) { console.log('Navigate to:', path); // Handle navigation in your app }, onStartTour: function() { console.log('Start application tour'); // Start your app's tour functionality } }); </script>

Method 3: Auto-initialization with Data Attributes

The easiest way to embed the widget with configuration via HTML attributes:

<div data-apptega-navigator data-width="400" data-height="600" data-user-id="user_123" data-organization-id="org_456" data-theme="light"> </div> <script src="http://localhost:3000/embed.js"></script>
